API are how you interact with your data, in addition to the auto-generated [REST API](/docs/04-01-api__rest-api.md), you have access server-side to the rime api which let you apply changes on your data.

## Rest API

```ts
const { docs } = await fetch('/api/pages')
```

## Rime API

```ts
import { error, type ServerLoadEvent } from '@sveltejs/kit';

export const load = async ({ locals }: ServerLoadEvent) => {
	const { rime } = locals;
	const nav = await rime.area('nav').find();
	return { nav };
};
```
